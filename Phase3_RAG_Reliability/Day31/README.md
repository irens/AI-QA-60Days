## 📚 2. 核心理论

### 2.1 关键词检索原理
```
用户查询 → 分词 → 倒排索引查询 → 相关性评分 → 结果排序
```

### 2.2 BM25算法核心
```
BM25(q, d) = Σ IDF(q_i) * (f(q_i, d) * (k1 + 1)) / (f(q_i, d) + k1 * (1 - b + b * |d| / avgdl))
```
- **IDF**: 逆文档频率，衡量词语独特性
- **k1**: 控制词频饱和度（通常1.2-2.0）
- **b**: 控制文档长度归一化（通常0.75）

### 2.3 关键词检索适用场景
| 场景类型 | 示例 | 为什么适合 |
|---------|------|-----------|
| 精确ID查询 | "订单号#12345" | 完全匹配 |
| 专有名词 | "BERT-base-uncased" | 术语精确 |
| 代码片段 | "import tensorflow" | 符号保留 |
| 版本号 | "v2.1.0" | 格式固定 |
